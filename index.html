Como publicar seu portfólio

Depois de gerar o arquivo HTML, veja como disponibilizá-lo online gratuitamente.

1

Salve o arquivo HTML

Com o portfólio aberto na nova aba, pressione Ctrl+S (ou Cmd+S no Mac) e salve o arquivo como index.html em uma pasta no seu computador.

2

Crie um repositório no GitHub

Acesse github.com, crie uma conta (se ainda não tiver) e clique em New repository. Dê um nome como meu-portfolio e marque como Public.

3

Envie o arquivo para o repositório

Dentro do repositório criado, clique em Add file → Upload files e arraste o index.html para lá. Confirme com Commit changes.

4

Ative o GitHub Pages

Vá em Settings → Pages do repositório. Em Branch, selecione main e clique em Save. Em alguns minutos, seu portfólio estará disponível em seuusuario.github.io/meu-portfolio.

5

Alternativa: Vercel (ainda mais rápido)

Acesse vercel.com, conecte sua conta do GitHub e importe o repositório. O Vercel detecta o arquivo automaticamente e publica em segundos, com um link personalizado.

💡 Domínio próprio: Tanto o GitHub Pages quanto o Vercel permitem conectar um domínio próprio (ex: anaclara.dev) gratuitamente. Basta comprar o domínio em um registrador e seguir as instruções de DNS de cada plataforma.
